1. システムの概要

🎯 このシステムでできること

出社状況管理システムは、あなたの勤務予定を管理し、チーム全体の出社状況を確認できるシステムです。

💡 重要なポイント
  • 月次プランナーで翌月の予定を事前に申請
  • 管理者が承認すると、出社状況と個人ページに反映
  • 当日の急な変更は出社状況ページで直接変更可能

🌐 3つの主要ページ

  1. 📊 出社状況ページ - チーム全体の今日の状況確認・当日の予定変更
  2. 📅 月次プランナー - 翌月の予定を事前に申請
  3. 👤 個人ページ - 自分の予定確認・履歴閲覧

2. ログイン方法

ログイン手順

  1. ブラウザ（Chrome または Edge推奨）を開く
  2. システムのURL を入力してアクセス
  3. メールアドレスとパスワードを入力
  4. 「ログイン」ボタンをクリック
📸 ログイン画面のスクリーンショット
（メールアドレス・パスワード入力欄、ログインボタン）
💡 ワンポイント

パスワードを忘れた場合は、「パスワードを忘れた方はこちら」リンクから再設定できます。

3. 📊 出社状況ページの使い方

ログイン後に最初に表示される画面です。チーム全体の今日の出社状況が一目で分かります。

📱 画面の見方

📸 出社状況メイン画面のスクリーンショット
（全体レイアウト、タイムライン、スタッフ一覧）

🎨 色分けの意味

緑色 = 出社・オンライン
青色 = リモート・在宅
オレンジ = 会議・外出
赤色 = 休暇・不在
紫色 = 研修・特別業務

🖱️ 基本操作

時間軸の移動

  1. 横スクロールで時間軸を左右に移動
  2. 今日ボタンで現在時刻にジャンプ
  3. マウスホイールでも横スクロール可能

表示の絞り込み

  1. 部署フィルターで特定の部署のみ表示
  2. グループフィルターで更に細かく絞り込み
  3. 「すべて」を選択すると全員表示に戻る

✏️ 当日の予定変更方法

⚡ 緊急時の予定変更

急な体調不良や会議の追加など、当日の予定変更はここで行います。

📝 予定の追加・変更手順

  1. 画面上で自分の名前の行を見つける
  2. 予定を追加したい時間帯をクリック
  3. 予定編集画面が開く
  4. ステータス（出社、リモート、休暇など）を選択
  5. 開始時間と終了時間を設定
  6. 必要に応じてメモを入力
  7. 保存ボタンをクリック
📸 予定編集モーダルのスクリーンショット
（ステータス選択、時間設定、メモ入力、保存ボタン）

🗑️ 予定の削除

  1. 削除したい予定をクリック
  2. 予定編集画面の削除ボタンをクリック
  3. 確認画面で「はい」をクリック
💡 便利な機能
  • プリセット機能で、よく使う予定パターンを素早く設定
  • ドラッグ&ドロップで予定の時間をかんたん調整
  • 右クリックでメニューを表示（ブラウザによる）

4. 📅 月次プランナーの使い方

翌月の予定を事前に申請するページです。管理者が承認すると、正式な予定として登録されます。

📋 月次プランナーの目的

来月の休暇や研修、出張などの予定を事前に申請することで、チーム全体のスケジュール調整がスムーズになります。

🌐 ページへの移動

  1. 出社状況ページの右上にある📅 月次プランナーをクリック
  2. 月次プランナーページが開きます
📸 月次プランナー画面のスクリーンショット
（月表示カレンダー、月切り替えボタン、申請ボタン）

📅 月の切り替え

  1. <ボタンで前の月
  2. >ボタンで次の月
  3. 今月ボタンで現在の月に戻る

✏️ 予定の申請方法

📝 予定追加の手順

  1. 申請したい日付をクリック
  2. 予定作成画面が開く
  3. ステータスを選択（休暇、研修、出張など）
  4. 時間を設定（終日の場合は9:00-18:00など）
  5. 理由・メモを入力（例：「年次休暇」「客先訪問」）
  6. 保存ボタンをクリック

📤 申請の送信

  1. すべての予定を入力後、画面上部の申請送信ボタンをクリック
  2. 確認画面で内容を確認
  3. 「送信」をクリックして申請完了

📊 申請状況の確認

グレー（点線） = 申請中（承認待ち）
色付き（実線） = 承認済み
赤色（太線） = 却下
💡 申請のコツ
  • 早めの申請：月末までに翌月分を申請
  • 詳細なメモ：理由を明確に記載
  • 複数日の連続休暇：まとめて申請すると管理が楽

5. 👤 個人ページの使い方

自分の予定確認と過去の勤務履歴を確認できるページです。

🌐 ページへの移動

  1. 出社状況ページの右上にある👤 個人ページをクリック
  2. 個人ページが開きます
📸 個人ページ画面のスクリーンショット
（個人カレンダー、月切り替え、予定詳細表示）

📅 表示の切り替え

  1. 月表示：1ヶ月の予定を俯瞰
  2. 週表示：1週間の詳細スケジュール
  3. 月切り替え：前月・翌月の履歴確認

📊 確認できる情報

承認済み予定

  • 月次プランナーで申請して承認された予定
  • 管理者が直接登録した予定
  • 基本契約時間（通常の勤務時間）

勤務統計情報

  • 月の総勤務時間
  • 休暇取得日数
  • 出社・リモート割合
💡 履歴活用法
  • 勤務実績確認：月末の時間確認
  • 休暇残日数把握：計画的な休暇取得
  • 勤務パターン分析：自分の働き方の振り返り

6. 🔄 基本的な使い方の流れ

1. 月次プランナー
翌月の予定を申請
→
2. 管理者承認
申請内容を確認・承認
→
3. 自動反映
出社状況・個人ページに表示

📋 毎月のルーティン

🗓️ 月末（25日頃）

  1. 月次プランナーで翌月の予定を入力
  2. 休暇、研修、出張などを申請
  3. 申請内容を確認して送信

📅 月初（1日〜3日）

  1. 承認結果を個人ページで確認
  2. 却下された項目があれば再申請
  3. 承認された予定が出社状況に反映されているか確認

🔄 日常業務

  1. 毎朝出社状況ページでチームの状況確認
  2. 急な予定変更は当日に直接変更
  3. 重要な変更は上司・チームに連絡
⚠️ 重要な注意点
  • 当日の変更は出社状況ページで（緊急時のみ）
  • 事前の予定は月次プランナーで申請
  • 長期休暇は早めに申請（1ヶ月前推奨）

7. ❓ よくある質問

Q1. 申請した予定が承認されません。どうすればいいですか？
A1. 以下を確認してください：
  • 申請内容に不備がないか（理由・時間など）
  • 他のメンバーと重複していないか
  • 会社の休暇規定に沿っているか
問題がない場合は、直接管理者に確認しましょう。
Q2. 予定を間違って入力しました。修正できますか？
A2. 状況によって異なります：
  • 申請前：月次プランナーで自由に修正可能
  • 申請後・承認前：一度取り消して再申請
  • 承認後：管理者に修正依頼、または当日に出社状況ページで変更
Q3. 急に体調不良になりました。どこで休暇申請すればいいですか？
A3. 当日の急な変更の場合：
  1. 出社状況ページで当日の予定を「休暇」に変更
  2. メモ欄に「体調不良」など理由を記入
  3. 上司・チームにも別途連絡
Q4. パスワードを忘れてしまいました。
A4. ログイン画面で：
  1. 「パスワードを忘れた方はこちら」をクリック
  2. メールアドレスを入力
  3. 送信されたメールの指示に従ってパスワード再設定
メールが届かない場合は、システム管理者に連絡してください。
Q5. 画面が正しく表示されません。
A5. 以下を試してください：
  1. ページ更新：F5キーまたはブラウザの更新ボタン
  2. キャッシュクリア：Ctrl+Shift+Delete でキャッシュ削除
  3. ブラウザ変更：Chrome または Edge を使用
  4. 再ログイン：一度ログアウトして再ログイン

8. 🔧 トラブル対応

🌐 ブラウザ関連

推奨ブラウザ

  • Google Chrome（最新版）
  • Microsoft Edge（最新版）
  • Internet Explorer は非対応

よくある問題と解決法

⚠️ 画面が崩れて表示される
  1. Ctrl + F5 でページを強制更新
  2. ブラウザのズーム倍率を100%に設定
  3. 広告ブロッカーを一時無効化
⚠️ 操作が反応しない
  1. 他のタブやアプリを閉じてメモリを確保
  2. ブラウザを再起動
  3. PCを再起動
⚠️ ログインできない
  1. メールアドレス・パスワードの入力ミスを確認
  2. CapsLockが有効になっていないか確認
  3. 「パスワードを忘れた方はこちら」でリセット
  4. それでも解決しない場合は管理者に連絡

🆘 困ったときの連絡先

📞 サポート連絡先
  • システム関連：ITサポート部門
  • 利用方法：直属の上司または人事部
  • 緊急時：総務部

※ 具体的な連絡先は会社の規定に従ってください
